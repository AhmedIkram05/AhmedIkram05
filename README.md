## 🔭 Featured Projects

<a href="https://github.com/AhmedIkram05/StockLens"><img src="https://img.shields.io/badge/StockLens_Mobile_App-View_Project-3776AB?style=for-the-badge&logo=github&logoColor=white&labelColor=000000"></a>  
Full-stack React Native app with Firebase auth, real-time data sync, and an OCR pipeline that converts physical receipts into structured financial records. Built in TypeScript across a clean data/logic/UI architecture.

<a href="https://github.com/AhmedIkram05/movie-recommendation-system"><img src="https://img.shields.io/badge/CineMatch_Recommendation_System-View_Project-47A248?style=for-the-badge&logo=github&logoColor=white&labelColor=000000"></a>  
Hybrid recommendation engine (collaborative filtering + content-based) built with Python, Scikit-learn, and Flask. Achieves ~0.75 Hit Rate and ~0.22 Precision@10 on the MovieLens dataset. Deployed as an interactive web app.

<a href="https://github.com/AhmedIkram05/haggis-predictive-modeling"><img src="https://img.shields.io/badge/Haggis_Data_Mining-View_Project-F7931E?style=for-the-badge&logo=github&logoColor=white&labelColor=000000"></a>  
End-to-end ML pipeline: EDA, feature engineering, XGBoost and Random Forest modelling, hyperparameter tuning. Achieved ~90% classification accuracy. Results validated with F1-scores and cross-validation.

<a href="https://github.com/AhmedIkram05/rental-car-company"><img src="https://img.shields.io/badge/Rental_Car_System_-View_Project-2496ED?style=for-the-badge&logo=github&logoColor=white&labelColor=000000"></a>  
Modular C++ console application using polymorphism, templates, and smart pointers. Features a fuzzy search engine built with Levenshtein distance, file persistence, and a full test suite.

<a href="https://github.com/AhmedIkram05/snapledger"><img src="https://img.shields.io/badge/SnapLedger_PWA-View_Project-EE4C2C?style=for-the-badge&logo=github&logoColor=white&labelColor=000000"></a>  
Offline-first personal finance PWA with client-side ML inference via TensorFlow.js. Smart expense categorisation, spending predictions, and anomaly detection — all running locally in the browser.
